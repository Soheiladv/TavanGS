راهنمای ایجاد قالب محاسباتی

این فرم برای ایجاد قالب‌های محاسباتی استفاده می‌شود. هر قالب شامل اسکیمای ورودی، خروجی و تنظیمات پیش‌فرض است. برای راهنمای کامل هر فیلد، روی آیکون کلیک کنید.

اطلاعات پایه

  • • نام: نام منحصر به فرد و قابل فهم
  • • نوع جلسه: نوع محاسبه (بودجه، امنیت، عملکرد، ...)
  • • توضیحات: شرح کامل قالب و کاربرد آن

نیازمندی‌های سیستم

  • • زمان تخمینی: مدت زمان اجرا (ثانیه)
  • • حافظه: مقدار RAM مورد نیاز (MB)
  • • نوع کاربر: حداقل سطح دسترسی
  • • CPU-Intensive: محاسبات سنگین پردازنده

JSON Schema

  • • Input Schema: ساختار ورودی‌های مورد نیاز
  • • Output Schema: ساختار خروجی‌های تولید شده
  • • Default Config: تنظیمات پیش‌فرض قالب
